# Entregas

Crea aquí **tu carpeta** con tu nombre o usuario de GitHub y coloca dentro tu documento único `auditoria-chatbot-ia.md`:

```
entregas/
└── nombre-apellido/            # o tu usuario de GitHub
    └── auditoria-chatbot-ia.md
```

Si haces el bonus de **Presidio** o de **Ollama**, puedes añadir en tu misma carpeta los archivos de apoyo (scripts, capturas, transcripciones), pero el análisis debe quedar dentro del documento único.

Puedes partir de la plantilla [`plantilla-auditoria-chatbot-ia.md`](plantilla-auditoria-chatbot-ia.md) (opcional).
